    华为eNSP 关闭Log

    数通
    发布时间 : 2021-08-17 15:38
    阅读 :

    华为的模拟器eNSP默认对变更会有Log产生，非常影响操作。关闭的方法如下：

    info-center source DS channel 0 log state off  trap state off
